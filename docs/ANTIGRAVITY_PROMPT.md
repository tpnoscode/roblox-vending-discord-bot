# Google Antigravity 초기 지시문

이 GitHub 저장소는 로블록스 아이템 및 로벅스 판매용 Discord 자판기 봇의 **초기 세팅 저장소**입니다. 현재 목표는 기능 전체 구현이 아니라, 이후 개발자가 빠르게 코딩을 시작할 수 있도록 저장소 구조와 배포 준비를 정리하는 것입니다.

## Antigravity에게 요청할 작업

아래 문장을 Antigravity에 그대로 입력합니다.

> 이 저장소를 기준으로 Discord 자판기 봇 개발 환경을 초기화해줘. 지금 단계에서는 실제 상품 지급, 자동 충전, 랜덤박스 확률 로직을 구현하지 말고, GitHub 기반 개발·검토·배포 준비가 가능한 구조를 유지해줘. `CLAUDE.md`와 `docs/SETUP_CHECKLIST.md`를 우선 읽고, 변경 사항은 작은 단위로 커밋해줘. 24시간 실행은 추후 Cloud Run, VPS, 또는 지속 실행 서버 중 하나로 연결할 수 있게 Dockerfile과 환경 변수 기준만 준비해줘.

## Antigravity 연결 후 확인할 항목

Antigravity가 저장소를 인식하면 다음 순서로 확인합니다.

| 순서 | 확인 항목 | 완료 기준 |
| --- | --- | --- |
| 1 | GitHub 저장소 연결 | 저장소 파일 목록이 Antigravity에서 보입니다. |
| 2 | `CLAUDE.md` 인식 | AI 작업 규칙으로 해당 파일을 참조합니다. |
| 3 | 환경 변수 설정 화면 확인 | Discord, 결제, Roblox 관련 변수 이름을 등록할 수 있습니다. |
| 4 | 배포 대상 선택 | 24시간 실행 대상이 Cloud Run, VPS, 또는 지속 실행 서버 중 하나로 정해집니다. |
| 5 | 첫 변경 사항 커밋 | 초기 설정 변경이 GitHub에 푸시됩니다. |
